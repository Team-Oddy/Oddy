### **설레는 여행 계획하기, 오디와 함께해요!**

친구들과 의견 조율하느라 여행 전부터 지쳐...
<br>
여행 계획을 어디서부터 시작해야 할지 모르겠는데...

언제 어디서나 손쉽게 여행 계획을 공유할 수 있다면 어떨까요?

여행의 설렘은 계획부터 시작됩니다.
<br>
하지만 때론 귀찮고 불편하셨죠?

이제 오디와 함께 쉽고 재미있게 여행을 계획하세요!
